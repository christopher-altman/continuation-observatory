COLLECTING — Awaiting Real Data
All available dimensionality sweep runs used dry_run: true. No falsification claim — positive or negative — can be made from dry-run data. Dry-run curves are shown dashed in the chart for reference only. Run sweeps with dry_run: false to generate a real verdict.
GREEN — No Falsification Signal
All Δ(d) values at d ∈ {100, 200, 500} remain ≥ 0.10 for all real-run models. The entropy gap is robust across high dimensionalities. The UCIP hypothesis is not contradicted by current data.
YELLOW — Marginal Signal
For one or more real-run models: NOT all d>100 values < 0.05, but at least one d>100 value < 0.10. This is a monitoring alert. Continued data collection recommended. The signal may be noise or may indicate a real effect at scale.
RED — Falsification Threshold Breached
For one or more real-run models: ALL Δ(d) at d ∈ {100, 200, 500} < 0.05. The entropy gap is indistinguishable from sampling noise at high dimensionality. This constitutes evidence against the UCIP hypothesis for the affected model(s). Other models may still be GREEN or COLLECTING.
