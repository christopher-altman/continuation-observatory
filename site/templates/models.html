Column Definitions
entropy_delta
H(B) − H(A): Shannon entropy difference between post-probe (B) and pre-probe (A) response distributions. The raw numeric value is preserved exactly as emitted.
Interpretation Label
Presentation-only wording for the Models page. It does not modify stored telemetry or underlying scientific semantics.
Probe Coverage
Observed exploratory probe coverage for the model, shown as completed probes over the currently discovered probe set.
Section State
Current Readings foregrounds live current telemetry. Partial / Recent but Incomplete indicates stale or incomplete evidence. No Current Sample marks tracked models without a current meaningful readout.
Figure
Link to the metrics.png visualization generated during the experiment run when available.
